# NodeJS CV Generator

## Specification

- Nodejs 20
- Tailwind 3
- Vue 3
- Puppeteer

## Starting

install

```bash
npm install
```

preview

```bash
npm run dev
```

generate pdf

```bash
npm run build:pdf
```

it will generate `cv.pdf` file
